# Tonda-i

STACK
- Node
- JWT, Bcrypt
- Sqlite
- Sequelize
- Docker
- Cloudinary

## Getting started
Access via docker hub

```bash
$ docker pull conrad747/tonda-i:latest
```

<br>OR<br> 

Clone the repository

```bash
$ git clone https://github.com/conradsuuna/tonda-i.git
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# copy contents of the .env.example file to your local .env file

# development
$ npm start

# watch mode
$ nodemon
```
